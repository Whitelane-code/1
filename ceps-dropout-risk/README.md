# CEPS Dropout Risk Prediction (LightGBM + SHAP)

本仓库提供一个可复现的机器学习建模流程，用于基于（本地）表格数据预测初中阶段学生的辍学风险（或类似“风险/事件”二分类任务），并输出 SHAP 可解释性结果，作为社会工作与教育干预场景中的**辅助决策支持工具**。

> 研究背景（开题方向）：困境青少年辍学风险预测与社区社工早期干预框架（LightGBM + SHAP + 场景映射）。  
> 本仓库开源的是**建模代码、配置与复现流程**，方便他人在自己的真实数据场景中复用与改进。

---

## 数据说明（重要）
本仓库不提供 CEPS 原始数据或清洗后的数据文件（如 `train.csv` / `test.csv`），主要原因包括但不限于：
- **数据授权与使用协议限制**：CEPS 数据通常需要在特定授权范围内使用，不适合在公开仓库二次分发；
- **隐私与合规考虑**：教育与青少年相关数据具有敏感性，即使做过处理，也应尽量避免公开传播可能引发的隐私风险；
- **学术规范**：遵循数据使用的合规边界，仅公开建模流程与可复现实验代码。

因此，仓库仅开源 **建模代码、配置文件与复现流程**。请使用你已获得合法授权的数据，或使用你自行收集与整理的真实数据完成复现。

---

## 推荐使用“真实数据”复现的原因
本项目的目标是让模型在社会工作实务中具有可落地性（例如社区社工站的早期筛查、风险分层与资源分配支持）。为了避免“只在论文里有效”，强烈建议你使用：
- 你在实务中合法获取的个案/服务记录（脱敏后）；
- 社区/学校的风险筛查问卷数据；
- 你自行设计并收集的访谈量表或追踪数据；
- 或你所在机构允许使用的公开/共享数据集。

你可以把这些数据整理成类似结构的表格（特征列 + 标签列），替换 `data/processed/train.csv` 与 `data/processed/test.csv` 即可运行本仓库代码。

---

## 社会工作领域的应用定位（借鉴思路）
本仓库的建模流程可用于“风险识别 → 可解释分析 → 干预决策支持”的实践链条，例如：
- **风险识别**：用 LightGBM 对个体进行风险预测（如辍学风险、心理风险、家庭压力风险等）；
- **可解释分析**：用 SHAP 输出关键风险因素，辅助社工开展评估与访谈聚焦；
- **干预分层**：依据风险等级（高/中/低）匹配个案工作、小组工作、社区预防性服务；
- **资源配置**：在社工资源有限时，为“优先介入对象”提供数据支持。

> 注意：模型输出应作为“辅助决策工具”，不能替代专业评估与伦理审查；实务使用需严格遵守隐私保护、知情同意与数据安全要求。
